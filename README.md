## 基于 Nestjs 9 的 react-admin backend

### 技术栈

    nestjs9.0
    typeorm
    reids
    yml
    mysql
    swagger

### 框架基础功能

    注册(完成)
    登录(完成)
    jwt认证(完成)(利用nestjs/jwt生成jwt，利用redis完成jwt自动过期和踢人功能)
    rabc(待完成)
    全局用户信息(待完成)

### 业务基础功能

    用户(完成)
    字典(完成)
    菜单(待完成)

### 启动命令

    pnpm run start:dev
